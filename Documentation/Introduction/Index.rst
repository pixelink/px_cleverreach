What does it do?
================

This extension uses the Cleverreach API to create newsletters in Cleverreach.
Simply create a new Typo3 Page with your desired newsletter stylings, add content elements in the main column and create the newsletter in backend module. HTML and Plaintext version will be created.

No newsletter subscription in this extension. Use either extension formhandler_cleverreach or create your form html in cleverreach and include on your website with the HTML content element.
